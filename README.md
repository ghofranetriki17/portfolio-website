# Portfolio Personnel

Site web personnel responsive présentant mon parcours académique, mes compétences techniques et mes projets.

## Description

Ce projet est un site web portfolio développé dans le cadre du module **Programmation Web I** (1ère année Génie Informatique) à l'Institut International de Technologie (IIT). Il présente de manière structurée mon profil professionnel, mes compétences et mon parcours académique.

## Fonctionnalités

Le site comprend les sections suivantes :

- **Accueil** : Présentation générale avec navigation et liens vers les réseaux sociaux
- **Parcours académique** : Diplômes obtenus avec présentation chronologique
- **Compétences** : Technologies et outils maîtrisés (langages, frameworks, SGBD, méthodes agiles)
- **Expériences** : Stages et expériences professionnelles (lieu, durée, objectifs)
- **Formations** : Manifestations scientifiques et événements
- **Projets** : Travaux pratiques et projets réalisés durant le cursus
- **A propos** : Passions, hobbies et qualités personnelles
- **Quiz interactif** : Quiz avec calcul automatique du score et affichage des corrections
- **Contact** : Formulaire de contact
- **Témoignage** : Retour d'expérience sur l'IIT

## Technologies utilisées

## Installation

1. Cloner le repository
   ```bash
   git clone https://github.com/ghofranetriki17/portfolio-website.git
   ```

2. Accéder au répertoire
   ```bash
   cd portfolio-website
   ```

3. Ouvrir `index.html` dans un navigateur ou utiliser une extension Live Server

## Points techniques

- Utilisation des balises sémantiques HTML5 (header, nav, main, section, article, footer)
- Design responsive adapté aux différentes tailles d'écran
- Quiz interactif avec validation et calcul de score en JavaScript
- Formulaires avec validation côté client
- Feuille de style CSS externe centralisée
